## Библиотеки компонентов

### 1. ShadCN UI
- **URL**: https://ui.shadcn.com
- **Что**: ~50 компонентов: кнопки, формы, модалки, таблицы, sidebar, карточки
- **Фишка**: Компоненты скачиваются в проект — **полный контроль над кодом**. 85.5K+ звёзд GitHub — стандарт 2025
- **Стек**: React, TypeScript, Tailwind CSS, Radix UI
- **Цена**: Бесплатно (MIT)

### 2. Aceternity UI
- **URL**: https://ui.aceternity.com
- **Что**: **200+ анимированных компонентов**: Aurora-фоны, Background Beams, 3D-карточки, Spotlight, параллакс, Bento Grid, текстовые анимации
- **Фишка**: Лучшая библиотека для «вау-эффектов» на лендингах
- **Стек**: React, Next.js, Tailwind CSS, Framer Motion
- **Цена**: Freemium — большинство бесплатны, Pro для премиум-шаблонов

### 3. Magic UI
- **URL**: https://magicui.design
- **Что**: **150+ анимированных компонентов**: Marquee, Hero Video Dialog, Dock-навигация, Shimmer Button, анимированный текст, градиенты
- **Фишка**: Идеальный компаньон для ShadCN
- **Стек**: React, TypeScript, Tailwind CSS, Motion
- **Цена**: Freemium

### 4. 21st.dev
- **URL**: https://21st.dev
- **Что**: Открытый реестр: кнопки, модалки, карточки, навигация, hero-секции, формы, хуки
- **Фишка**: **Magic Chat — AI-ассистент** для генерации компонентов. Установка через `npx shadcn`. Интеграция с Cursor и Windsurf
- **Стек**: React, Next.js, Tailwind CSS, Radix UI
- **Цена**: Бесплатно + Pro

### 5. Origin UI
- **URL**: https://originui.com
- **Что**: Copy-paste компоненты от Cal.com: кнопки, формы, селекты, диалоги, таблицы
- **Фишка**: Построен на **Base UI** (не Radix). Трёхуровневая архитектура: Primitives → Particles → Atoms. Оптимизирован для AI-ассистентов
- **Стек**: React, TypeScript, Tailwind CSS
- **Цена**: Бесплатно

### 6. DaisyUI
- **URL**: https://daisyui.com
- **Что**: Крупнейший набор Tailwind CSS компонентов: кнопки, карточки, модалки, навигация, формы, hero, stats
- **Фишка**: **35+ встроенных тем** включая dark mode. Не требует JavaScript — чистый CSS. Фреймворк-агностик
- **Стек**: Tailwind CSS (любой фреймворк)
- **Цена**: Бесплатно (MIT)

### 7. HeroUI (бывший NextUI)
- **URL**: https://heroui.com
- **Что**: Полнофункциональные React-компоненты
- **Фишка**: На **React Aria** (лучшая accessibility). Sub-40KB gzipped. Фокус на производительности и семантическом HTML
- **Стек**: React, Tailwind CSS
- **Цена**: Бесплатно

### 8. Mantine
- **URL**: https://mantine.dev
- **Что**: **100+ компонентов + 50+ хуков**: формы с валидацией, уведомления, rich text editor, чарты, command palette
- **Фишка**: Мощная TypeScript-поддержка, полноценная экосистема
- **Стек**: React, TypeScript
- **Цена**: Бесплатно (MIT)

## Расширенные библиотеки

### 9. Tailwind Plus (бывший Tailwind UI)
- **URL**: https://tailwindcss.com/plus
- **Что**: **500+ профессиональных компонентов**: hero, pricing, testimonials, навигация, формы. Шаблоны на Next.js
- **Фишка**: От создателей Tailwind CSS. Код для React, Vue и HTML. Пожизненные обновления
- **Стек**: React, Vue, HTML, Tailwind CSS
- **Цена**: $299 (personal), $979 (team)

### 10. Flowbite
- **URL**: https://flowbite.com
- **Что**: **600+ компонентов**: кнопки, модалки, формы, datepicker, карусели, чарты. 53+ шаблонов страниц
- **Фишка**: Мультифреймворковая поддержка: React, Vue, Svelte, Angular. WCAG-доступность
- **Стек**: React, Vue, Svelte, Angular, Tailwind CSS
- **Цена**: Freemium, Pro: $299

### 11. Preline UI
- **URL**: https://preline.co
- **Что**: **640+ компонентов + 220+ блоков** + 5 бесплатных шаблонов. Figma Design System
- **Фишка**: Agent Skills для Cursor, Claude Code, Gemini CLI
- **Стек**: HTML, React, Vue, Tailwind CSS
- **Цена**: Freemium, Pro: ~$99/год

### 12. TailGrids
- **URL**: https://tailgrids.com
- **Что**: **600+ React-компонентов**: hero, navbar, pricing, testimonials, CTA
- **Фишка**: CLI-инструмент, Figma Design System. Полная экосистема
- **Стек**: React, Tailwind CSS, Next.js
- **Цена**: Freemium

## Анимационные библиотеки

### 13. Motion Primitives
- **URL**: https://motion-primitives.com
- **Что**: Анимированные компоненты: Text Effects (reveal, scramble, gradient), Spotlight, Dock, hover-tilts, параллакс
- **Фишка**: Идеален для деликатных анимаций, совместим с ShadCN
- **Стек**: React, Tailwind CSS, Framer Motion
- **Цена**: Freemium

### 14. React Bits
- **URL**: https://reactbits.dev
- **Что**: **110+ анимированных компонентов**: BlurText, GradientText, Aurora-фоны, StarField
- **Фишка**: **#2 в JS Rising Stars 2025** (37K+ звёзд). Многие анимации на чистом CSS (без Framer Motion). Совместим с React Server Components
- **Стек**: React, CSS
- **Цена**: Бесплатно

### 15. Hover.dev
- **URL**: https://www.hover.dev
- **Что**: Интерактивные секции: Hero, FAQ, Features, Pricing, Kanban, 3D-секции, навбары, карусели
- **Фишка**: Фокус на **готовых секциях**, а не отдельных элементах
- **Стек**: React, Tailwind CSS, Framer Motion
- **Цена**: Freemium

### 16. Motion (ex Framer Motion)
- **URL**: https://motion.dev
- **Что**: Полноценная анимационная библиотека: spring-физика, layout-анимации, жесты, AnimatePresence
- **Фишка**: **30M+ npm-загрузок/мес** — стандарт анимации для React. Теперь фреймворк-агностичный (JS, Vue, React)
- **Стек**: React, Vue, JS
- **Цена**: Бесплатно, Motion+ — платная подписка

## Маркетплейсы

### 17. UIverse
- **URL**: https://uiverse.io
- **Что**: Крупнейшая открытая библиотека UI-элементов от сообщества: светящиеся кнопки, карточки, формы, переключатели, лоадеры
- **Фишка**: Pinterest-стиль, copy-paste CSS. Фреймворк-агностик
- **Цена**: Бесплатно

### 18. Framer Marketplace
- **URL**: https://www.framer.com/marketplace/
- **Что**: Шаблоны сайтов, UI-компоненты, плагины для Framer. Категории: SaaS, portfolio, agency, blog
- **Фишка**: Drag-and-drop, встроенные анимации
- **Цена**: Бесплатные и платные ($10–$99+)

### 19. Segment UI
- **URL**: https://segmentui.com
- **Что**: **1000+ вариаций** стилей, компонентов, интеракций для Framer и Figma
- **Фишка**: CMS-интеграции, кастомные генераторы кода. Экономит **40–60% времени** для Framer-фрилансеров
- **Цена**: Freemium

### 20. Float UI
- **URL**: https://floatui.com
- **Что**: Чистые Tailwind-компоненты: hero, pricing, навбары, формы, карточки, footer
- **Фишка**: Мультифреймворковая поддержка: React, Vue, Svelte, HTML. Чистый минималистичный код
- **Цена**: Бесплатно

## AI-инструменты

### 21. Google Stitch
- **URL**: https://stitch.withgoogle.com
- **Что**: AI-генерация UI из текстовых промптов или скетчей. Дизайн-системы, многостраничные прототипы
- **Фишка**: Экспорт в **Figma и HTML/CSS**. Извлечение дизайн-системы из любого URL. Gemini 2.5 Pro. До 350 генераций/мес
- **Цена**: Бесплатно (Google Labs)
- **Источники**: https://uxpilot.ai/blogs/google-stitch-ai | https://www.mindstudio.ai/blog/how-to-use-google-stitch-website-design-system | https://uxplanet.org/google-stitch-for-ui-design-544cf8b42d52

## Анимации и 3D

### 22. LottieFiles
- **URL**: https://lottiefiles.com
- **Что**: **500 000+ Lottie-анимаций** для веба и мобайла. Lottie Creator, Lottie Editor
- **Фишка**: Плагины для Figma, Webflow, Framer, Canva. **16 млн+ пользователей**. Лёгкие JSON-анимации
- **Цена**: Freemium, от ~$20/мес

### 23. GSAP (GreenSock)
- **URL**: https://gsap.com
- **Что**: Мощнейшая JS-библиотека: timeline-контроль, **ScrollTrigger**, SplitText, Draggable, MotionPath, MorphSVG
- **Фишка**: Стандарт индустрии для сложных анимаций. Совместим с React, Vue, Three.js, Webflow
- **Цена**: Freemium (платные: ScrollSmoother, SplitText, MorphSVG)

### 24. Spline
- **URL**: https://spline.design
- **Что**: Браузерный 3D-редактор: интерактивные 3D-сцены, анимации, продуктовые визуализации
- **Фишка**: Экспорт в Web, React, iframe. **Самый простой способ добавить 3D** без программирования. Real-time collaboration. Интеграция с Webflow и Framer
- **Цена**: Freemium, от $9/мес

### 25. Three.js / React Three Fiber
- **URL**: https://threejs.org / https://r3f.docs.pmnd.rs
- **Что**: Фундамент **95% всех 3D-сайтов**. 100K+ звёзд GitHub. Полноценная 3D-графика
- **Фишка**: React Three Fiber — декларативный wrapper. Огромная экосистема: Drei (100+ хелперов), физика (Rapier), постэффекты
- **Цена**: Бесплатно (MIT)
