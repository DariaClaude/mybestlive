# SEO vs GEO: Полный гайд

---

## 1. Чем отличается SEO от GEO

**SEO** — статья попадает в поисковую выдачу (Google, Яндекс).
Человек вводит запрос → видит список сайтов → кликает → читает.

**GEO** — статья попадает в ответ AI (ChatGPT, Perplexity, Claude).
Человек спрашивает AI → AI цитирует твою статью как источник.

| | SEO | GEO |
|---|---|---|
| Кто читает первым | Поисковый алгоритм | AI-модель |
| Что важно | Ключевые слова, ссылки | Чёткие факты, структура |
| Результат | Позиция в выдаче | Цитата в AI-ответе |

---

## 2. Как написать статью для поисковиков

### Контент
- **H1** — содержит основной запрос дословно
- **Мета-описание** — 1–2 предложения: о чём статья и что получит читатель
- **Подзаголовки H2/H3** — разбивают текст, включают слова из темы
- **Alt-текст у картинок** — поисковик читает только его
- **Текст для человека** — отвечает на вопрос, без искусственных повторов ключей
- **Внешние ссылки** — хотя бы одна ссылка с другого сайта = сигнал доверия

### Техника
- **Schema Markup** — разметка Article + FAQ (Rank Math / Yoast — автоматически)
- **Canonical URL** — страница доступна строго по одному адресу
- **Дата обновления** — видимая в тексте + тег `dateModified` в метаданных
- **Индексация** — нет тега `noindex`, страница открыта для поисковиков
- **Sitemap** — добавлен в Google Search Console и Яндекс Вебмастер
- **Скорость загрузки** — PageSpeed Insights > 70 баллов на мобильном

---

## 3. Как написать статью для AI-выдачи (GEO)

### Структура
- Прямой ответ в первых 2–3 предложениях — AI берёт именно его
- Q&A-блок в конце статьи — вопрос жирным, ответ сразу под ним
- Короткие абзацы — 1 абзац = 1 мысль
- Заголовки-утверждения: не *"О продукте"* → *"Что такое X и как это работает"*

### Контент
- Определения терминов прямо в тексте — AI их цитирует как дефиниции
- Конкретные цифры, даты, факты со ссылкой на источник
- Сравнения: *"X отличается от Y тем, что..."* — любимый паттерн AI
- Полное раскрытие темы — AI оценивает исчерпанность, не плотность ключей

### Фразы, которые AI цитирует чаще всего
> *"По данным [источник], X составляет Y%"*
> *"[Термин] — это [определение в одном предложении]"*
> *"Шаг 1: ... Шаг 2: ... Шаг 3: ..."*
> *"Главное отличие X от Y: [одно предложение]"*

### Доверие
- Имя автора, должность, опыт — указаны в тексте
- Бренд упомянут рядом с экспертным утверждением
- Ссылки на первичные источники: исследования, статистика, официальные данные

## Чеклист перед публикацией

**SEO:**
- [ ] Запрос есть в H1
- [ ] Мета-описание заполнено
- [ ] Есть подзаголовки H2
- [ ] У картинок прописан alt
- [ ] Страница открыта для индексации
- [ ] Canonical URL единый
- [ ] Schema Markup настроена (Rank Math / Yoast)
- [ ] Дата обновления видна в тексте
- [ ] Sitemap добавлен в Search Console / Вебмастер
- [ ] Скорость сайта > 70 (PageSpeed Insights)

**GEO:**
- [ ] Прямой ответ в первых 2–3 предложениях
- [ ] Есть Q&A-блок
- [ ] Абзацы короткие, каждый = одна мысль
- [ ] Есть определения терминов
- [ ] Есть цифры и ссылки на источники
- [ ] Указан автор с экспертизой
- [ ] Бренд упомянут в экспертном контексте
